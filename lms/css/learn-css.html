Introduction to CSS

CSS (Cascading Style Sheets) is the language used to style and layout web pages. It enhances the appearance of HTML by controlling the look and feel of elements like colors, fonts, spacing, and positioning.

CSS Structure

CSS is structured using rulesets, which consist of selectors and declarations. A selector targets the HTML elements you want to style, and declarations define the styling properties and values.

            selector {
                property: value;
            }

For example, the following CSS code styles all paragraphs to be blue:

            p {
                color: blue;
            }

CSS Selectors

CSS selectors are used to select the HTML elements you want to style. The basic types of selectors include element selectors, class selectors, and ID selectors.

  • element: Selects all instances of a specific HTML element, like p for paragraphs. Example: p { color: red; }
  • .class: Selects all elements with a specific class. Example: .heading { font-size: 24px; }
  • #id: Selects an element with a specific ID. Example: #main-header { background-color: yellow; }

CSS Box Model

The CSS Box Model is a fundamental concept that describes how elements are structured on a webpage. Every element in CSS is considered a rectangular box, which consists of four parts: content, padding, border, and margin.

  • Content: The actual content of the box, such as text or an image.
  • Padding: The space between the content and the border, creating inner space within the box.
  • Border: The edge around the padding, which can be styled with different widths, colors, and styles.
  • Margin: The space outside the border, creating distance between this box and other elements.
            /* Example CSS for Box Model */
            .box {
                width: 300px;
                padding: 20px;
                border: 5px solid black;
                margin: 10px;
            }

CSS Layout

CSS layout techniques allow you to control the arrangement of elements on a webpage. Common layout methods include block, inline, flexbox, and grid.

  • Block Layout: Elements take up the full width available, such as div and p tags. They stack vertically.
  • Inline Layout: Elements only take up as much width as necessary, such as span and a tags. They stack horizontally.
  • Flexbox: A layout model that allows elements to align and distribute space within a container dynamically. Example: display: flex;
  • Grid Layout: A two-dimensional layout model that allows for both rows and columns. Example: display: grid;
            /* Example CSS for Flexbox Layout */
            .container {
                display: flex;
                justify-content: space-between;
            }
            
            /* Example CSS for Grid Layout */
            .grid-container {
                display: grid;
                grid-template-columns: 1fr 1fr;
                gap: 10px;
            }

CSS Flexbox

CSS Flexbox is a powerful layout module that allows for the alignment and distribution of space among items in a container, even when their size is unknown. It provides a more efficient way to design flexible, responsive layouts.

  • flex-container: The parent element where Flexbox is applied. Example: display: flex;
  • flex-item: The child elements within the container, which automatically adjust based on the container’s rules.
  • justify-content: Controls the horizontal alignment of items within the container.
  • align-items: Controls the vertical alignment of items within the container.
            /* Example CSS for Flexbox */
            .container {
                display: flex;
                justify-content: center;
                align-items: center;
                height: 100vh;
            }
            .item {
                padding: 10px;
                background-color: lightblue;
            }
