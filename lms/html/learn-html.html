Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the basic structure for web content by defining elements like headings, paragraphs, images, and links.

HTML Structure

HTML documents are structured using elements like <html>, <head>, and <body>. These elements define the layout of the webpage and organize its content.

                      <!DOCTYPE html>
                      <html>
                          <head>
                              <title>Page Title</title>
                          </head>
                          <body>
                              <h1>This is a Heading</h1>
                              <p>This is a paragraph.</p>
                          </body>
                      </html>

HTML Elements

HTML elements are the building blocks of HTML pages. Each element is represented by a tag, such as <h1> for headings or <p> for paragraphs.

  • <h1></h1>: Defines a top-level heading.
  • <p></p>: Defines a paragraph.
  • <a></a>: Defines a hyperlink.
  • <img>: Defines an image.

HTML Attributes

HTML attributes provide additional information about HTML elements. Attributes are always included in the opening tag and consist of a name and a value, such as href or src.

<a href="https://example.com">This is a link</a>
<img src="image.jpg" alt="A description of the image">

Links and Anchors

HTML links are created using the <a> tag. Links are used to navigate between different pages, documents, or sections of a website.

<a href="https://example.com">Visit Example</a>

You can also create anchors within the same page using the id attribute.

<a href="#section">Go to Section</a>
<div id="section">This is the section</div>

Lists

HTML provides two types of lists: ordered lists (<ol>) and unordered lists (<ul>).

<ul>
    <li>First item</li>
    <li>Second item</li>
</ul>

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

HTML Tables

Tables in HTML are created using the <table> element, along with <tr> for rows and <td> for cells.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
</table>

HTML Forms

Forms allow users to input data and send it to a server. HTML forms are created using the <form> element, and input fields are created with tags like <input> and <textarea>.

<form action="/submit" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <input type="submit" value="Submit">
</form>
