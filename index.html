LEARNING HTML

INTRO TO HTML

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It's the backbone of a website, providing the structure and content that the web browser renders to the user. HTML consists of a series of elements, represented by tags (<>), which are used to define the different parts of a web page.

HTML TEXT FORMATTING

HTML provides various elements to format text on a web page. Headings (h1-h6) are used to define headings, paragraphs (p) for body text, and bold (b or strong) and italic (i or em) for emphasis. Lists (ul, ol, li) are used to create unordered and ordered lists. Text alignment can be controlled using center, left, and right attributes.

HTML Capabilities

HTML is capable of creating web pages with structured content, including headings, paragraphs, images, and links. It can also embed multimedia content like audio and video. However, HTML has limitations, such as not being a programming language, which means it can't create dynamic content or interact with users.

HTML Navigation and Links

HTML uses anchor tags (a) to create hyperlinks, which allow users to navigate between web pages. The href attribute specifies the link destination. Links can be internal (within the same website), external (to other websites), email, or phone links. Navigation menus are created using unordered lists (ul) and list items (li).

HTML Working with Graphics and Images

HTML uses the img tag to add images to web pages. The src attribute specifies the image source, and the alt attribute provides alternative text for accessibility. Common image formats include jpg, png, and gif. Graphics can be created using svg and canvas elements.

HTML Working with Media

HTML5 introduced the audio and video tags, making it easy to embed multimedia content. The audio tag supports mp3 and other formats, while the video tag supports mp4 and webm. Embedding media from other sources can be done using iframe and embed tags.

HTML Content Identification

HTML5 introduced semantic elements to define the structure of a web page. The header element defines the header section, footer defines the footer, and section defines a self-contained section. Article elements represent independent content, and nav elements define navigation sections.

HTML Integration

HTML is often used with CSS (Cascading Style Sheets) for styling and layout, and JavaScript for interactivity and functionality. HTML5 features like semantic elements, multimedia, and canvas enhance the functionality of web pages.

HTML Working with Forms

HTML forms allow users to interact with web pages. The form tag defines a form, input fields (input) allow user input, and labels (label) provide text for input fields. Form validation attributes like required and pattern ensure user input meets specific criteria.

HTML Interactive Elements

HTML provides various interactive elements, including buttons (button), dropdowns (select, option), checkboxes (input type="checkbox"), and radio buttons (input type="radio"). These elements allow users to interact with web pages.

1. Intro to CSS

CSS (Cascading Style Sheets) is a styling language used to control the layout and appearance of web pages. It separates presentation from structure, allowing for more flexibility and maintainability. CSS is applied to HTML elements using selectors, properties, and values.

Adding CSS Selectors

CSS selectors target specific HTML elements for styling. Types of selectors include:

  • Element selectors (e.g., p, h1)
  • Class selectors (.class)
  • ID selectors (#id)
  • Attribute selectors ([attribute])
  • Pseudo-class selectors (:hover, :active)
  • Pseudo-elements selectors (::before, ::after)

CSS Images and Colors

CSS is used to add images and control colors on web pages.

  • Background images: background-image, background-repeat, background-position
  • Foreground images: img, src
  • Colors: color, background-color, hex codes, RGB, RGBA

CSS Boxes, Types

CSS boxes model HTML elements as rectangular boxes.

  • Block-level elements (div, p): occupy full width, stack vertically
  • Inline elements (span, a): occupy only necessary width, stack horizontally

CSS Boxes, Sizes

CSS controls box sizes using:

  • Width and height properties
  • Box-sizing property (content-box, padding-box, border-box)
  • Margin and padding properties
  • Border property

Advanced CSS Properties and Concepts

Advanced CSS topics include:

  • Positioning: absolute, relative, fixed, sticky
  • Floating: float, clear
  • Display: inline-block, flex, grid
  • Transitions and animations
  • Media queries for responsive design
